# Problem 03

Write a C program to input elements into an array and sort the array elements in ascending order.

Input
<pre>
Input size of array: 10
Input array elements: 20, 2, 10, 6, 52, 31, 0, 45, 79, 40
</pre>

Output
<pre>
Array sorted in ascending order: 0, 2, 6, 10, 20, 31, 40, 45, 52, 79
</pre>
