# Problem 12

Write a C program that takes one number as input from the user and prints its multiplication table using a loop.

## Examples
<b>Input<b><br>
<pre>Enter a number: 2</pre>
<b>Output<b><br>
<pre>Multiplication table of 2:
2 × 1 = 2
2 × 2 = 4
2 × 3 = 6
2 × 4 = 8
2 × 5 = 10
2 × 6 = 12
2 × 7 = 14
2 × 8 = 16
2 × 9 = 18
2 × 10 = 20</pre>
