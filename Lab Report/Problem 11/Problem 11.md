# Problem 11

Write a C program to check whether a string is a palindrome or not without using a ‘for’ loop.


## Examples
<b>Input<b><br>
<pre>madam</pre>
<pre>hello</pre>
<pre>mom</pre>
<pre>dad</pre>

<b>Output<b><br>

<pre>"madam" is a palindrome.</pre>
<pre>"hello" is not a palindrome.</pre>
<pre>"mom" is a palindrome.</pre>
<pre>"eye" is a palindrome.</pre>
<pre>"dad" is a palindrome.</pre>
